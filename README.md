# Songwriting Collaboration Platform

## Overview
A commercial-grade SaaS marketing website template for a songwriting collaboration platform.

## Features
- Modern UI/UX with Glassmorphism
- Dark/Light Theme Switching
- Responsive Bootstrap 5 Layout
- GSAP Animations

## Setup
1. Open `index.html` in your browser.
2. Ensure you have an internet connection for CDN assets (Bootstrap, Google Fonts, GSAP).
